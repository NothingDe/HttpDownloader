# 目录说明

1. CustomWidgets 一些通用的窗口，比如自定义Windows窗口等。
2. CustomControls 一些基础空间和组合控件。
3. HttpDowner 是App项目
- HttpDownloader Http下载核心类实现
- UIDownloader 主界面类
- UIConfigDialog 配置界面类
